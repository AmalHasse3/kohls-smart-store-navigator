# Kohl’s Smart Store Navigator

Interactive retail navigation prototype created for the Kohl’s Emerging Talent Summit 2026.

## Overview
The Kohl’s Smart Store Navigator is designed to improve the in-store and digital shopping experience by helping customers easily navigate departments, discover products, and engage with Kohl’s rewards and lifestyle experiences.

This concept supports the “Kohl’s LifeStage Rewards” ecosystem by connecting Sephora shoppers, Babies R Us families, and Kohl’s customers through one seamless retail experience.

---

## Features
- Interactive store map
- Product and department search
- Smart route navigation
- Sephora @ Kohl’s integration
- Babies R Us integration
- Kohl’s Café concept
- Rewards and customer service integration
- Personalized shopping experience

---

## Key Goals
- Increase customer engagement
- Improve in-store convenience
- Encourage cross-category shopping
- Increase loyalty app usage
- Create a modern retail experience
